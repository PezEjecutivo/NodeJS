# Task Tracker CLI

Es un proyecto realizado con NodeJS utilizando exclusivamente sus funciones nativas, sin importar librerias o dependencias externas.

El proyecto esta hecho con los requisitos y requerimientos de la pagina: https://roadmap.sh/projects/github-user-activity

Es importante aclarar, que la API ha cambiado y ahora para obtener información que menciona el proyecto es necesario un Token

## Comandos disponibles

En este proyecto no hay comandos como tal, solamente puedes pasarle un argumento, el cúal es un nombre de usuario.

`node ./github-activity.js <username> `
